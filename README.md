# 🚀 Miranda Gourmet Food

Miranda Gourmet Food es una compañía de catering corporativo premium. Lleva más de 12 años en la industria, comprometida siempre con brindar experiencias memorables y asegurar la satisfacción de sus clientes.

## 📖 Historia del Desarrollo

Este proyecto nació de la necesidad de renovar la página web antigua y dar una experiencia de usuario excepcional. A lo largo del proceso, el equipo se enfocó en crear una experiencia de usuario fluida, enfrentando retos técnicos como la integración con pasarelas de pago, la creación de un carrito de compras, y la estandarización de productos.
Hitos Principales
1. Fase inicial: Concepción y Diseño: Definición de la arquitectura y creación de prototipos
2. Fase de desarrollo: Estructuración: Implementación de la base técnica y componentes atómicos
3. Fase de integración: Conexión con APIs y lógica de negocio

## 🛠️ Stack Tecnológico

| Tecnología | Propósito |
| -- | -- |
| React | Framework principal |
| Bootstrap + CSS | Estilizado y diseño responsivo |
| TypeScript | Tipado estático para mayor robustez |
| Vite | Herramienta de construcción (Bundler) |

## 🏗️ Arquitectura del Proyecto

El desarrollo se basó en una estructura de carpetas modular para facilitar el mantenimiento y la escalabilidad:
- src/components: Componentes reutilizables de la interfaz
- src/hooks: Lógica personalizada de React
- src/services: Llamadas a la API y manejo de datos externos
- src/context: Proveedores de estado global

## 🧠 Desafíos y Aprendizajes

Durante el desarrollo, nos encontramos con situaciones clave que marcaron el rumbo del código:
- Optimización de Rendimiento: Implementamos lazy loading para mejorar los tiempos de carga inicial
- Accesibilidad: Nos aseguramos de que la navegación fuera compatible con lectores de pantalla y teclado
- Decisiones de Diseño: Se optó por un enfoque "Mobile-First" debido a que la narrativa del usuario indicaba un uso mayoritario en dispositivos móviles
